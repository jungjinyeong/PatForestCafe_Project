*음식 제조*
 - 음식을 넣고 조합하는 방법
 - 음식은 최대 5가지를 넣을 수 있음


음식 레시피는 알려주지 않음

음식에는 힌트가 있음

[비비빅 게임 참고]
*도토리 재배[커피]*
 - 도토리를 재배[도토리를 재배하는 일꾼 고용으로 인한 골드 소모]
   -> 고용한 일꾼 인원에 따라 추가 값 상승
   -> 일꾼 레벨업 기능 추가[레벨 업에 따른 골드 소모] -> 레벨업 시 더 많은 도토리를 재배
   -> 일꾼 오픈 레벨 제한[콘텐츠 소모 속도 조절]

*우유 가공*
 - 우유를 재배[젖소 구매로 인한 골드 소모]
   -> 소지한 젖소 인원에 따라 추가 값 상승
   -> 젖소 레벨업 기능 추가[레벨 업에 따른 골드 소모]-> 레벨 업 시 더 많은 우유를 제공
   -> 젖소 오픈 레벨 제한[콘텐츠 소모 속도 조절]

*이벤트 발생!*
이벤트 발생 조건 : 충돌 이벤트 중 확률적으로 발생 & 비둘기가 날아와 특수 이벤트 발생[선택제]
이벤트 내용 : 메뉴 추천 -> 캐릭터에 따른 메뉴[체질에 맞는 음료 추천] ->
             Ex) 우유를 먹으면 배가 아픈데 시원한 음료가 마시고 싶어요! -> 우유가 안들어간 아이스 메뉴

- 결과 -
이벤트 성공 시 = 기존 메뉴 수입 n% ↑
이벤트 실패 시 = 기존 메뉴 수입 n% ↓ [Ex)20%, 아메리카노 1잔 = 1000원 → 성공 시 1200원, 실패 시 800원]

* 손님 홍보 *

* 배달 시스템 *
- 발생 조건 : 하루의 퀘스트 같은 형식으로 제공
시스템 내용 : 이벤트 발생 -> 이벤트 내용과 같은 형식으로 음식을 제조 후, 보낸 경우 n시간 후 복귀 시 많은 양의 보상 제공
